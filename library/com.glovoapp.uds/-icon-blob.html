Design System IconBlob

IconBlob is a composable that draws an icon inside a blob. The icon is centered inside the blob.

Parameters

icon

The icon to draw inside the blob.

style

The colors of the icon blob.

size

The size of the icon blob.

modifier

The modifier to be applied to the icon blob.

contentDescription

The content description of the icon blob.

blobShape

The shape of the blob.
