@Composable

Design System Icon Button Component

Icon Button with an Icon or a CircularProgressIndicator in the center.

Parameters

icon

sets the icon based on Icon to be shown when the button is not in a loading state. Only the foreground resource is used.

contentDescription

optional content description for accessibility.

modifier

the modifier is applied to the internal ButtonSurface usage.

style

is used to define the background and other behaviors of each button variant.

size

IconButtonSize is responsible for the size of the button, the icon, padding, and border width.

enabled

controls the enabled state of this button. When false, this component will not respond to user input, and it will appear visually disabled and disabled to accessibility services.

loading

whether the button is in a loading state. If true, the button will display a circular progress indicator and hide the icon.

interactionSource

the MutableInteractionSource representing the stream of Interactions for this button. You can create and pass in your own remembered instance to observe. Interactions and customize the appearance/behavior of this button in different states.

onClick

called when this button is clicked.
