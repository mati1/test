Badge

@Composable
fun Badge(style: BadgeStyle, size: BadgeSize, modifier: Modifier = Modifier, count: String? = null)

Design System Badge Component

Parameters

style

BadgeStyle define the background color to be used in the badge

count

for BadgeSize.S and BadgeSize.M the badge supports a count text, to values bigger than 99 its recommended usage of 99+ as a count value

size

both badge and dot contains Medium and Small variants

modifier

the modifier to be applied to the Badge component.
